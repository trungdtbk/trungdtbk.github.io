CREATE A SIMPLE SCRATCH APP

Mar 23, 2020

This tutorial shows you how to create your first ever project in Scratch. A Scratch project is an application (or app for short) which can be anything from game to music video or an animated story. The tutorial will also explain what makes up an application in Scratch. Let’s get started.

Read more about Scratch project here

To create a Scratch project, open Scratch (you will need an account if you want to save and share your work), then select Create.

Get to know Scratch User Interface

The Scratch interface will look like the image below. The main areas of the interface are Instruction repository, coding window, sprite window, backdrops and preview window.

What is a Sprite?

In each project, you can many Sprites. A Sprite is kind of a character (or player) in your app. You will need to give each Sprite a sequence of instructions so that the Sprite will do thing you want it to do. Grab instructions from the Instruction repository on the left and place them in the Coding window. Instructions are sticked together in a top-down manner. When the app starts, it will run instruction by instruction from the top to the bottom and causes the Sprite to do things.

A Sprite can have many different costumes. For instance, one Sprite can have a Cat costume while the other has a Dog one. Or the same Sprite can a standing cat and a sitting cat costumes. Click on the Costumes tab to see what you have and to add, delete or modify costumes.

What are instructions?

Blocks in the code repository are instructions, for instance say [hello] and move (10) steps blocks. You see them visually as blocks but they will be translated into the computer code so that the computer can execute them.

For most blocks, to khow what it does click on it and see the effect in the preview window.

What is a backdrop?

Each Scratch app has a stage which can have one or many more backdrops. You can program the stage almost the same way as you do with the Sprites. For instance, you can put codes to the stage to switch from the sunny backdrop to rainy one.
